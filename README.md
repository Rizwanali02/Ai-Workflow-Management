# AI-Powered Workflow Management System

A modern, scalable workflow management system built with Next.js and enhanced with AI capabilities to improve productivity, collaboration, and task management.
 
---

## 🚀 Features

### 🧠 Smart Dashboard

* Role-based dashboards for Admin, Manager, and Employee
* AI-generated workflow summaries using Gemini AI

### 📁 Project & Task Management

* Create and manage projects with detailed views
* Task lifecycle: Pending → In Progress → Completed
* AI-powered task description refinement
* Timeline and task distribution visualization

### 👥 Team Collaboration

* Task-level comments and discussion threads
* Image attachments (JPG/PNG supported)
* Real-time notifications using Socket.io

### 🔐 Security & Access Control

* Role-Based Access Control (RBAC)
* Module-level permissions
* Protected routes for secure navigation

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS
* **UI Components:** Shadcn UI, Lucide Icons, 
* **Backend:** Node.js, Next.js API Routes
* **Database:** MongoDB (Mongoose)
* **Authentication:** JWT (JSON Web Tokens)
* **State Management:** React Context API
* **HTTP Client:** Axios
* **Real-time:** Socket.io
* **AI Integration:** Google Gemini Pro SDK

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Rizwanali02/Ai-Workflow-Management

# Navigate to project folder
cd Ai-Workflow-Management

# Install dependencies
npm install

# Run development server
npm run dev
```
